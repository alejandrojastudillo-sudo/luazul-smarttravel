⚙️ Configuración inicial

Seguí estos pasos para conectar el panel con tu Google Sheets. Solo lo hacés una vez.

1

Crear el Google Sheet

Abrí sheets.google.com y creá una hoja nueva llamada Luazul Smart Travel Clientes. Creá una pestaña llamada Clientes con estas columnas exactas en la fila 1:

email | password | nombre | apellido | documento | domicilio | telefono1 | telefono2 | destino | paquete | personas | noches | fecha_viaje | total_usd | cuotas_total | cuotas_pagadas | monto_cuota | estado | contrato_url | voucher_url

2

Publicar como API (Apps Script)

En el Sheet: Extensiones → Apps Script. Borrá todo y pegá el código que aparece abajo. Luego hacé clic en Implementar → Nueva implementación → Aplicación web. Ejecutar como: Yo. Acceso: Cualquier persona. Copiá la URL que te da.

function doGet(e) {
  var email = e.parameter.email;
  var pass = e.parameter.password;
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Clientes');
  var data = sheet.getDataRange().getValues();
  var headers = data[0];
  for (var i = 1; i < data.length; i++) {
    var row = {};
    headers.forEach(function(h,j){ row[h] = data[i][j]; });
    if (row.email == email && row.password == pass) {
      return ContentService.createTextOutput(JSON.stringify({ok:true,data:row})).setMimeType(ContentService.MimeType.JSON);
    }
  }
  return ContentService.createTextOutput(JSON.stringify({ok:false})).setMimeType(ContentService.MimeType.JSON);
}
3

Pegar la URL aquí abajo

Pegá la URL de tu Apps Script deployment para activar el panel real.

4

Agregar clientes al Sheet

Completá una fila por cliente. Campos obligatorios: email, password, nombre, destino, total_usd, cuotas_total, cuotas_pagadas, monto_cuota. El campo estado puede ser: activo, completado.
